🚀 Dub Editor App 调用示例

提示：此页面演示如何从 Web 页面通过 Deep Link 打开 Dub Editor App。

方式 1: 直接跳转（推荐）

输入编辑器地址，点击按钮直接打开 App：

方式 2: 优雅降级（检测是否安装）

✅ 正在打开 App...

方式 3: 使用隐藏 iframe（更平滑）

📝 代码示例

方式 1: 直接跳转

function openEditor(editorUrl) {
  const deepLink = `dubeditor://open?url=${encodeURIComponent(editorUrl)}`;
  window.location.href = deepLink;
}

方式 2: 优雅降级

function openEditor(editorUrl) {
  const deepLink = `dubeditor://open?url=${encodeURIComponent(editorUrl)}`;
  const downloadUrl = 'https://your-domain.com/download';

  window.location.href = deepLink;

  // 2 秒后如果还在页面，说明没安装 App
  setTimeout(() => {
    if (confirm('检测到您未安装 Dub Editor App，是否前往下载？')) {
      window.location.href = downloadUrl;
    }
  }, 2000);
}

方式 3: 使用隐藏 iframe

function openEditor(editorUrl) {
  const deepLink = `dubeditor://open?url=${encodeURIComponent(editorUrl)}`;
  const downloadUrl = 'https://your-domain.com/download';

  // 创建隐藏的 iframe
  const iframe = document.createElement('iframe');
  iframe.style.display = 'none';
  iframe.src = deepLink;
  document.body.appendChild(iframe);

  // 检测是否成功打开
  const start = Date.now();

  setTimeout(() => {
    const elapsed = Date.now() - start;

    // 如果时间差小于 2100ms，说明没有离开页面
    if (elapsed < 2100) {
      if (confirm('检测到您未安装 Dub Editor App，是否前往下载？')) {
        window.location.href = downloadUrl;
      }
    }

    // 清理 iframe
    document.body.removeChild(iframe);
  }, 2000);
}

React 组件示例

import { useState } from 'react';

function EditorButton({ editorUrl }) {
  const [showDownload, setShowDownload] = useState(false);

  const openEditor = () => {
    const deepLink = `dubeditor://open?url=${encodeURIComponent(editorUrl)}`;
    window.location.href = deepLink;

    setTimeout(() => {
      setShowDownload(true);
    }, 2000);
  };

  return (
    <div>
      <button onClick={openEditor}>
        在 App 中打开编辑器
      </button>

      {showDownload && (
        <div className="download-tip">
          <p>未检测到 App，<a href="/download">点击下载</a></p>
        </div>
      )}
    </div>
  );
}

🔗 URL Scheme 格式

dubeditor://open?url=https://your-domain.com/editor?id=123&token=abc

参数说明：

  • url - 编辑器完整 URL（必需，需要 URL 编码）
